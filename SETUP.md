# LinkedIn Agent — Partner Setup

## 1. Web App
No setup needed. Just go to:
**https://linked-in-marketing-agent.vercel.app**

Sign in with Google, complete the voice onboarding, and you're ready to create posts.

---

## 2. Chrome Extension

### Install
1. Download and unzip `linkedin-agent-extension.zip`
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** → select the `chrome-mv3` folder inside the unzipped folder

You should see the LinkedIn Agent icon appear in your Chrome toolbar.

### Use it
1. Sign in at the web app first (step 1 above)
2. Go to [linkedin.com](https://www.linkedin.com)
3. Click the **◀** tab on the right edge of the screen to open the sidebar
4. Type what you want to post about and hit **Send**

---

## Notes
- The extension only activates on `linkedin.com` pages
- If the sidebar says "not logged in", sign in at the web app and refresh LinkedIn
- Chrome may warn that the extension is unverified — this is expected for unpacked extensions, click **Keep** to proceed
